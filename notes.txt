Differences between let, const, var keywords:
----------------------------------------------
-> let keyword scope only inside of the block
-> let keyword it initialize the value only at once    

->Var keyword scope global (We access inside +outside)
->We can initialize multiple times 

->const keyword takes values in a constant mode only
  we can't update or modify const keyword values....





for(let a=0;a<5;a++)
{
    console.log("a value is="+a);
}
console.log("outside of the loop a value is="+a)





for(var a=0;a<5;a++)
{
    console.log("a value is="+a);
}
console.log("outside of the loop a value is="+a)


Note: Const variable cannot be modified

for(const a=0;a<5;a++)
{
    console.log("a value is="+a);
}
console.log("outside of the loop a value is="+a)
